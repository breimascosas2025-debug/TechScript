  LIBERACIÓN DE MEMORIA RAM - OPTIMIZADOR
  Mejora el rendimiento liberando memoria ocupada
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script libera memoria RAM que está siendo ocupada innecesariamente
por procesos del sistema, mejorando la velocidad de respuesta de tu PC.

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Libera el Working Set de procesos en ejecución
2. Limpia la caché del sistema de archivos
3. Reinicia Explorer.exe (libera recursos de interfaz)
4. Ejecuta el Garbage Collector de .NET
5. Optimiza prioridades de procesos
6. Libera memoria standby no utilizada

CUÁNDO USARLO:
───────────────────────────────────────────────────────────────────
- Cuando el equipo está lento
- Después de cerrar programas pesados (juegos, editores)
- Cuando la memoria RAM está al 80% o más
- Antes de ejecutar aplicaciones que requieren mucha RAM

INSTRUCCIONES DE USO:
───────────────────────────────────────────────────────────────────
1. Guarda tu trabajo abierto
2. Cierra programas que no necesites
3. Ejecuta el script como Administrador
4. Espera a que termine el proceso
5. Continúa usando tu PC (notarás mejoría)

⚠️ IMPORTANTE:
───────────────────────────────────────────────────────────────────
- El script reinicia Explorer (barra de tareas parpadea)
- Guarda tu trabajo antes de ejecutar
- Es normal que tarde 10-20 segundos
- Puedes ejecutarlo varias veces al día

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
