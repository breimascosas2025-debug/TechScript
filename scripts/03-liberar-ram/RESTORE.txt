⚠️ ESTE SCRIPT NO REQUIERE REVERSIÓN

Los cambios realizados son TEMPORALES y se ajustan automáticamente:
→ Windows gestiona la memoria RAM dinámicamente
→ Los procesos recuperan su memoria según necesiten
→ Las prioridades se ajustan por el sistema operativo

───────────────────────────────────────────────────────────────────
¿QUÉ CAMBIA EL SCRIPT?
───────────────────────────────────────────────────────────────────

1. Working Set de procesos
   → Reduce temporalmente el uso de memoria
   → Los procesos recuperan memoria automáticamente

2. Caché del sistema de archivos
   → Limpia archivos en caché no usados
   → Windows reconstruye la caché según necesidad

3. Reinicio de Explorer
   → Cierra y vuelve a abrir el explorador de Windows
   → Libera handles de ventanas cerradas

4. Prioridades de procesos
   → Cambios temporales, Windows las readjusta
