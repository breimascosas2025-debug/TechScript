  OPTIMIZACIÓN DE DISCO DURO/SSD
  Mejora el rendimiento del almacenamiento
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script detecta automáticamente si tu disco es HDD o SSD y
aplica las optimizaciones adecuadas para cada tipo de almacenamiento,
mejorando el rendimiento y la vida útil del disco.

OPTIMIZACIONES PARA SSD:
───────────────────────────────────────────────────────────────────
✓ Habilita comando TRIM
✓ Optimización SSD (no desfragmentación)
✓ Ajusta el indexador de búsqueda
✓ Desactiva Superfetch/Prefetch (innecesario en SSD)
✓ Programa comprobación de errores

OPTIMIZACIONES PARA HDD:
───────────────────────────────────────────────────────────────────
✓ Desfragmentación completa
✓ Limpieza de disco
✓ Habilita Superfetch/Prefetch
✓ Programa comprobación de errores

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Detecta el tipo de disco (SSD vs HDD)
2. Ejecuta limpieza de disco (cleanmgr)
3. Aplica TRIM (SSD) o desfragmenta (HDD)
4. Programa comprobación de errores (chkdsk)
5. Optimiza el indexador de búsqueda
6. Ajusta Superfetch/Prefetch según el tipo de disco

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Ejecuta como Administrador
2. Espera a que termine (puede tardar 10-30 min)
3. Reinicia el equipo
4. Notarás mejoras en velocidad de acceso a archivos

⚠️ NOTA:
───────────────────────────────────────────────────────────────────
- La desfragmentación en SSD no es necesaria y reduce su vida útil
- El script detecta automáticamente y aplica lo correcto
- Superfetch en SSD consume RAM innecesariamente

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
