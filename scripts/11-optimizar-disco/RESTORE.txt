✅ ALGUNOS CAMBIOS SON REVERTIBLES

───────────────────────────────────────────────────────────────────
REVERTIR CONFIGURACIÓN DE SUPERFETCH/PREFETCH:
───────────────────────────────────────────────────────────────────

1. Presiona Win + R, escribe: regedit
2. Navega a:
   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\PrefetchParameters
3. Cambia estos valores:
   - EnableSuperfetch: 1 (habilitar) o 0 (deshabilitar)
   - EnablePrefetcher: 1 (habilitar) o 0 (deshabilitar)
4. Reinicia el equipo

Valores recomendados:
- HDD: EnableSuperfetch=1, EnablePrefetcher=3
- SSD: EnableSuperfetch=0, EnablePrefetcher=0

───────────────────────────────────────────────────────────────────
REVERTIR INDEXADOR DE BÚSQUEDA:
───────────────────────────────────────────────────────────────────

1. Presiona Win + R, escribe: regedit
2. Navega a:
   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search
3. Cambia EnableIndexerBackOff a 0 o elimínalo
4. Reinicia el equipo

───────────────────────────────────────────────────────────────────
LIMPIEZA Y DESFRAGMENTACIÓN:
───────────────────────────────────────────────────────────────────

Estos procesos NO tienen reversión:
→ Limpieza de archivos elimina datos permanentemente
→ Desfragmentación reorganiza archivos (no daña)
→ TRIM es automático y no configurable por usuario

───────────────────────────────────────────────────────────────────
RECOMENDACIÓN
───────────────────────────────────────────────────────────────────
Los cambios de este script son generalmente BENEFICIOSOS.
Revertir no suele ser necesario ni recomendable.
