  LIMPIEZA TOTAL DE ARCHIVOS TEMPORALES v2.0
  Script de Optimización Avanzada para Windows
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script realiza una limpieza profunda y segura de todos los archivos
temporales del sistema Windows, liberando espacio en disco y mejorando
el rendimiento general.

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Elimina archivos de la carpeta %TEMP% del usuario
2. Limpia la carpeta C:\Windows\Temp
3. Borra archivos del Prefetch (caché de programas)
4. Limpia la caché de Windows Update
5. Elimina caché de Internet Explorer y Edge
6. Borra reportes de error de Windows (WER)
7. Vacía la papelera de reciclaje

INSTRUCCIONES DE USO:
───────────────────────────────────────────────────────────────────
1. Cierra todos los programas abiertos
2. Haz clic derecho en el archivo .bat
3. Selecciona "Ejecutar como administrador"
4. Espera a que el script termine (puede tomar 1-5 minutos)
5. Reinicia el equipo para completar la optimización

⚠️ IMPORTANTE:
───────────────────────────────────────────────────────────────────
- Cierra navegadores antes de ejecutar (se borrará caché)
- Guarda tu trabajo antes de iniciar
- Algunos archivos en uso no se eliminarán (esto es normal)
- No interrumpas el proceso una vez iniciado

VERSIÓN: 2.0
FECHA: 2024
COMPATIBILIDAD: Windows 10/11 (también funciona en Windows 7/8)
