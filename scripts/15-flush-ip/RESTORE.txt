⚠️ ESTE SCRIPT NO REQUIERE REVERSIÓN

El script NO hace cambios permanentes:
→ Libera IPs temporales
→ Renueva conexiones (obtiene nueva IP automáticamente)
→ Limpia cachés temporales

───────────────────────────────────────────────────────────────────
¿POR QUÉ NO SE PUEDE REVERTIR?
───────────────────────────────────────────────────────────────────

Dirección IP:
→ Tu router/ISP asigna IPs automáticamente
→ Obtienes la mejor IP disponible en cada momento
→ "Revertir" significaría volver a tener una IP potencialmente problemática

Caché DNS:
→ Información temporal que se regenera sola
→ Limpiarla es beneficioso, no perjudicial

───────────────────────────────────────────────────────────────────
SI NECESITAS CONFIGURACIÓN IP ESPECÍFICA:
───────────────────────────────────────────────────────────────────

Si usabas IP estática y quieres recuperarla:

1. Panel de control → Redes e Internet → Centro de redes
2. Cambiar configuración del adaptador
3. Clic derecho en tu conexión → Propiedades
4. Protocolo de Internet versión 4 (TCP/IPv4) → Propiedades
5. Selecciona "Usar la siguiente dirección IP"
6. Introduce tu IP estática, máscara y gateway
7. Aceptar

Si no recuerdas la configuración:
→ Contacta a tu administrador de red
→ O deja DHCP (IP automática) que es el estándar
