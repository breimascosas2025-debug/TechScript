  FLUSH IP Y RED - RESET COMPLETO
  Limpieza de configuración de red
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script realiza una limpieza completa de la configuración de red,
liberando direcciones IP, limpiando cachés y renovando todas las
conexiones para resolver problemas de conectividad.

PROBLEMAS QUE RESUELVE:
───────────────────────────────────────────────────────────────────
✓ Conexión a internet lenta
✓ Conexión intermitente que se cae
✓ Errores de resolución DNS
✓ Conflictos de direcciones IP
✓ "Conexión limitada o nula"
✓ Problemas después de cambiar de red
✓ Cambios de DNS no aplicados

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Libera dirección IPv4 actual
2. Libera dirección IPv6
3. Libera configuraciones alternativas
4. Limpia caché DNS completa
5. Limpia caché de resolución NetBIOS
6. Renueva configuración IPv4
7. Renueva configuración IPv6
8. Muestra la nueva configuración

RESULTADO:
───────────────────────────────────────────────────────────────────
- Obtienes una dirección IP completamente nueva
- Todas las cachés de red se limpian
- Las conexiones se establecen desde cero
- Resuelve conflictos de configuración

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Ejecuta como Administrador
2. Espera a que termine
3. Prueba tu conexión a internet
4. No es necesario reiniciar (pero puede ayudar)

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
