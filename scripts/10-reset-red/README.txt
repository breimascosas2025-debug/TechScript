  RESET DE RED COMPLETO
  Restaurar configuración de red a valores predeterminados
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script restablece completamente la configuración de red de
Windows a sus valores predeterminados, solucionando problemas de
conectividad complejos y persistentes.

PROBLEMAS QUE RESUELVE:
───────────────────────────────────────────────────────────────────
✓ Sin acceso a Internet
✓ Conexión limitada o sin conectividad
✓ No se puede acceder a páginas web
✓ Problemas de conexión VPN
✓ Errores de red después de malware
✓ Configuración de red corrupta
✓ Problemas de DNS persistentes

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Libera y renueva la dirección IP
2. Limpia la caché DNS
3. Restablece Winsock (API de sockets Windows)
4. Reinicia el stack TCP/IP
5. Restablece el firewall de Windows
6. Borra la tabla de rutas
7. Renueva configuración NetBIOS
8. Restablece configuración de proxy

⚠️ ADVERTENCIA:
───────────────────────────────────────────────────────────────────
- Se perderán configuraciones de red personalizadas
- Deberás reconfigurar VPNs después
- Configuraciones de proxy personalizadas se borrarán
- Si usas IP estática, deberás configurarla nuevamente

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Ejecuta como Administrador
2. Espera a que termine
3. REINICIA EL EQUIPO (obligatorio)
4. Verifica la conexión a Internet

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
