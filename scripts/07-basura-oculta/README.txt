  ELIMINACIÓN DE ARCHIVOS BASURA OCULTOS
  Limpieza profunda de archivos ocultos
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script encuentra y elimina archivos ocultos y de sistema que
acumulan espacio en disco sin que el usuario lo note, como archivos
de caché de miniaturas, metadatos de otros sistemas operativos, y
archivos vacíos.

¿QUÉ ELIMINA ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
✓ Thumbs.db - Archivos de caché de miniaturas de Windows
✓ .DS_Store - Archivos de metadatos de macOS
✓ desktop.ini - Archivos de configuración de carpetas
✓ Archivos vacíos (0 bytes)
✓ Archivos temporales de instalación (.tmp, .bak)
✓ Caché de iconos (IconCache.db)
✓ Archivos de log antiguos

ESPACIO LIBERADO TÍPICO:
───────────────────────────────────────────────────────────────────
- Sistemas con muchas imágenes: 100MB - 2GB
- Sistemas con carpetas compartidas macOS: 50MB - 500MB
- Uso general: 20MB - 200MB

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Cierra el Explorador de Windows (se reiniciará automáticamente)
2. Ejecuta como Administrador
3. Espera a que termine
4. Reinicio no requerido

⚠️ NOTA:
───────────────────────────────────────────────────────────────────
- Thumbs.db se regenerarán al navegar carpetas con imágenes
- .DS_Store se crean al acceder carpetas desde macOS
- desktop.ini se regeneran para configuración de carpetas
- Esto es NORMAL y no indica un problema

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
