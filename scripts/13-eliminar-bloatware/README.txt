  ELIMINACIÓN DE BLOATWARE BÁSICO
  Elimina apps preinstaladas innecesarias
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script elimina aplicaciones preinstaladas (bloatware) de Windows
que consumen recursos del sistema, espacio en disco y ralentizan el
inicio, pero que la mayoría de usuarios nunca utilizan.

APLICACIONES ELIMINADAS:
───────────────────────────────────────────────────────────────────
Aplicaciones Microsoft:
✓ 3D Builder, Alarms & Clock, Mail & Calendar
✓ My Office, Skype, Get Started (Get Help)
✓ Groove Music, Maps, Microsoft Solitaire
✓ Money, Movies & TV, News, OneNote
✓ People, Phone, Sports, Voice Recorder
✓ Weather, Xbox App

Apps de terceros (si están instaladas):
✓ Netflix, Spotify, Disney+
✓ McAfee, Norton (versiones de prueba)

También elimina:
✓ OneDrive (aplicación de escritorio)
✓ Microsoft Teams (versión personal)
✓ Sugerencias de la Tienda Windows
✓ Contenido promocional

⚠️ ADVERTENCIA:
───────────────────────────────────────────────────────────────────
- Las apps se eliminan para el usuario actual
- No afectan el funcionamiento del sistema
- Puedes reinstalarlas desde Microsoft Store si las necesitas
- OneDrive se desinstala completamente (backup primero)

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Haz backup de archivos de OneDrive si los usas
2. Ejecuta como Administrador
3. Espera a que termine
4. Reinicia el equipo

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
