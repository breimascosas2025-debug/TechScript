  DESACTIVACIÓN DE SERVICIOS INNECESARIOS
  Optimización Avanzada de Windows
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script desactiva servicios de Windows que consumen recursos pero
que no son esenciales para la mayoría de usuarios, mejorando el
rendimiento y liberando memoria RAM.

SERVICIOS QUE DESACTIVA:
───────────────────────────────────────────────────────────────────
✓ DiagTrack (Telemetría/Recopilación de datos)
✓ dmwappushservice (Servicio de notificaciones WAP)
✓ lfsvc (Servicio de geolocalización)
✓ MapsBroker (Administrador de mapas descargados)
✓ WbioSrvc (Servicios biométricos de Windows)
✓ XblAuthManager, XblGameSave, XboxNetApiSvc (Xbox)
✓ WSearch (Búsqueda de Windows)
✓ SysMain (Superfetch - pre-carga de programas)
✓ WerSvc (Informes de error de Windows)
✓ Spooler (Cola de impresión)
✓ Fax (Servicio de fax)
✓ TapiSrv (Telefonía)

⚠️ IMPORTANTE:
───────────────────────────────────────────────────────────────────
- Desactivar SysMain SOLO recomendado si tienes SSD
- Desactivar Spooler impide imprimir (reactívalo cuando necesites)
- Desactivar WSearch hace más lenta la búsqueda de archivos
- Los servicios de Xbox solo si no juegas en PC

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Lee RESTORE.txt antes de ejecutar
2. Ejecuta como Administrador
3. Reinicia el PC después
4. Si hay problemas, ejecuta REVERTIR_SERVICIOS.BAT

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
