  OPTIMIZACIÓN DE INICIO DE WINDOWS
  Acelera el arranque del sistema
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script optimiza el tiempo de inicio de Windows desactivando
programas y servicios que se ejecutan automáticamente al arrancar
el sistema, ralentizando el proceso de inicio.

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Desactiva OneDrive del inicio automático
2. Desactiva Cortana del arranque
3. Desactiva tareas de Office en segundo plano
4. Desactiva Skype del inicio
5. Desactiva tareas programadas de telemetría
6. Desactiva la hibernación (mejora inicio/apagado)

RESULTADO ESPERADO:
───────────────────────────────────────────────────────────────────
- Inicio de Windows más rápido
- Menos programas en la bandeja del sistema
- Menor uso de RAM al iniciar
- Apagado más rápido

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Ejecuta como Administrador
2. Espera a que termine el proceso
3. Reinicia tu PC para ver los cambios

PARA MÁS CONTROL:
───────────────────────────────────────────────────────────────────
Presiona Ctrl+Shift+Esc (Administrador de tareas)
→ Pestaña "Inicio"
→ Desactiva más programas según necesites

⚠️ NO desactives:
- Tu antivirus
- Drivers de audio/gráficos
- Software de hardware (teclado, mouse gaming)

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
