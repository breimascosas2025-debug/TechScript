  LIMPIEZA DE NAVEGADORES
  Limpieza de caché, cookies e historial
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script limpia automáticamente la caché, cookies, historial y
archivos temporales de todos los navegadores web instalados en el
sistema, liberando espacio en disco y mejorando la privacidad.

NAVEGADORES SOPORTADOS:
───────────────────────────────────────────────────────────────────
✓ Google Chrome
✓ Mozilla Firefox
✓ Microsoft Edge
✓ Opera
✓ Brave Browser
✓ Internet Explorer / Edge Legacy

¿QUÉ LIMPIA?
───────────────────────────────────────────────────────────────────
✓ Caché de páginas web (imágenes, archivos)
✓ Caché de código JavaScript
✓ Caché de GPU (aceleración gráfica)
✓ Cookies de sitios web
✓ Historial de navegación
✓ Archivos LOG y temporales
✓ Miniaturas de sitios

ESPACIO LIBERADO TÍPICO:
───────────────────────────────────────────────────────────────────
- Uso ligero: 50MB - 200MB
- Uso normal: 200MB - 1GB
- Uso intensivo: 1GB - 5GB+

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Guarda pestañas importantes (se cerrarán navegadores)
2. Cierra todos los navegadores
3. Ejecuta como Administrador
4. Espera a que termine
5. Reinicia navegadores (inicio de sesión puede requerirse)

⚠️ ADVERTENCIA:
───────────────────────────────────────────────────────────────────
- Se cerrarán todos los navegadores automáticamente
- Las sesiones abiertas se perderán
- Deberás iniciar sesión nuevamente en sitios web
- Las páginas cargarán más lento la primera vez

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11
