  Script Avanzado de Limpieza y Optimización

DESCRIPCIÓN:
Este es el script más completo de la colección. Ejecuta TODAS las
optimizaciones disponibles en un solo proceso automatizado,
incluyendo limpieza profunda, reparación del sistema y optimización.

¿QUÉ HACE ESTE SCRIPT?
═══════════════════════════════════════════════════════════════════

FASE 1 - PREPARACIÓN:
✓ Crea punto de restauración del sistema
✓ Cierra programas innecesarios
✓ Prepara servicios para optimización

FASE 2 - LIMPIEZA PROFUNDA:
✓ Elimina archivos temporales de todo el sistema
✓ Limpia caché DNS y red
✓ Elimina logs del sistema
✓ Borra archivos basura ocultos
✓ Limpia caché de navegadores
✓ Limpia Windows Update
✓ Vacía papelera de reciclaje

FASE 3 - REPARACIÓN:
✓ Verifica archivos del sistema (SFC)
✓ Repara imagen Windows (DISM)
✓ Restablece Winsock
✓ Renueva configuración de red

FASE 4 - OPTIMIZACIÓN:
✓ Libera memoria RAM
✓ Optimiza disco (SSD/HDD detectado automáticamente)
✓ Desactiva telemetría
✓ Optimiza inicio del sistema
✓ Reinicia servicios esenciales

TIEMPO DE EJECUCIÓN: 20-40 minutos
═══════════════════════════════════════════════════════════════════

INSTRUCCIONES:
───────────────────────────────────────────────────────────────────
1. Guarda TODO tu trabajo
2. Cierra programas importantes
3. Ejecuta como Administrador
4. NO CIERRES la ventana durante el proceso
5. El equipo se reiniciará automáticamente al final

⚠️ ADVERTENCIAS:
───────────────────────────────────────────────────────────────────
