  REPARACIÓN BÁSICA DEL SISTEMA
  Verificación y corrección de archivos Windows
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script ejecuta las herramientas oficiales de Windows para verificar
y reparar archivos del sistema corruptos o dañados, restaurando la
estabilidad del sistema operativo.

HERRAMIENTAS UTILIZADAS:
───────────────────────────────────────────────────────────────────
✓ SFC (System File Checker) - Verifica archivos protegidos
✓ DISM CheckHealth - Comprueba estado de la imagen Windows
✓ DISM ScanHealth - Escanea componentes del sistema
✓ DISM RestoreHealth - Repara daños en la imagen
✓ Análisis de Component Store - Verifica almacenamiento

PROBLEMAS QUE RESUELVE:
───────────────────────────────────────────────────────────────────
- Errores de archivos del sistema corruptos
- Pantallas azules relacionadas con archivos del sistema
- Funciones de Windows que no responden
- Actualizaciones que fallan al instalarse
- Lentitud inexplicable del sistema
- Errores de aplicaciones del sistema

TIEMPO DE EJECUCIÓN:
───────────────────────────────────────────────────────────────────
- SFC: 5-15 minutos
- DISM ScanHealth: 5-10 minutos
- DISM RestoreHealth: 10-20 minutos
- TOTAL ESTIMADO: 15-30 minutos
