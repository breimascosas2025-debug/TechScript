  FLUSH DNS - LIMPIEZA DE CACHÉ DNS
  Resolución de problemas de navegación web
═══════════════════════════════════════════════════════════════════

DESCRIPCIÓN:
Este script limpia la caché DNS de Windows y renueva la configuración
de red, solucionando problemas comunes de navegación web y conectividad.

¿QUÉ HACE ESTE SCRIPT?
───────────────────────────────────────────────────────────────────
1. Ejecuta ipconfig /flushdns para limpiar caché DNS
2. Renueva la dirección IP (release/renew)
3. Reinicia el servicio DNS Client
4. Muestra el estado actual de la resolución DNS

PROBLEMAS QUE RESUELVE:
───────────────────────────────────────────────────────────────────
- Páginas web que no cargan o cargan lentamente
- Error "DNS_PROBE_FINISHED_NXDOMAIN"
- Error "DNS_PROBE_FINISHED_BAD_CONFIG"
- Cambios de DNS no reflejados inmediatamente
- Problemas de resolución de nombres de dominio
- Conexión a internet lenta o intermitente

INSTRUCCIONES DE USO:
───────────────────────────────────────────────────────────────────
1. Cierra todos los navegadores web
2. Ejecuta el script como Administrador
3. Espera a que termine el proceso
4. Reinicia tu navegador

⚠️ IMPORTANTE:
───────────────────────────────────────────────────────────────────
- Cierra navegadores antes de ejecutar
- No es necesario reiniciar el equipo
- El proceso es seguro y reversible

VERSIÓN: 1.0
COMPATIBILIDAD: Windows 10/11/8/7
